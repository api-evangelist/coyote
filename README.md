# Coyote Logistics

Coyote Logistics is a global third-party logistics (3PL) provider and freight brokerage platform, now part of RXO, offering REST APIs for truckload capacity, less-than-truckload (LTL) shipping, freight quoting, shipment tracking, load management, customs brokerage, and supply chain visibility.

The CoyoteGO API platform serves both shippers and carriers through a REST API backed by OAuth 2.0 client credentials authentication. The API is hosted on Microsoft Azure API Management and provides access to real-time freight pricing, live tracking, and load management capabilities across a network of 100,000+ carriers.
